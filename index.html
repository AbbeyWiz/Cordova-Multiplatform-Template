Tabs navigation experiencie

  • Android

    • Tabs on top and filled background.
    • Only text, no icons.
    • Highlighted, white and stripped.

  • iOS

    • Tabs on bottom.
    • Icon on top and text.
    • Highlighted, filled with color.

  • Windows

    • Tabs on top.
    • Icon on top and text.
    • Highlighted, colored but not filled.
    • Swipe to change tab.

Slide panel and hamburguer

  • Android

    • White hamburguer.
    • Slide panel on right.

  • iOS

    • Colored hamburguer.
    • Slide panel on right.

  • Windows

    • With tabs navigation, hamburguer is a tab.
    • Slide panel is a page.

Back button

  • Android

    • White back button.
    • Hides hamburguer icon.

  • iOS

    • Colored back button.
    • Hides hamburguer icon.

  • Windows

    • Always with physical button.

Action buttons

  • Android

  • iOS

    • Colored action buttons.
    • Button on top.
    • Button on right.

  • Windows

    • Colored action buttons.
